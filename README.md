# Isle of Man Planning Statistics

Data analysis and statistics on planning applications across the Isle of Man, sourced from [PlanningPortal.im](https://planningportal.im) — the island's most comprehensive planning intelligence platform.

## What's Here

This repository contains statistical analysis, data breakdowns, and visualisations derived from over 81,000 planning applications submitted to the Isle of Man government.

### Topics Covered

- Approval rates by parish, application type, and case officer
- Application volume trends over time
- Most common application types (extensions, change of use, new builds)
- Processing times and decision patterns
- Planning agent performance data

## Data Source

All data is sourced from [PlanningPortal.im](https://planningportal.im), which aggregates and normalises planning data from the Isle of Man Department of Environment, Food and Agriculture (DEFA).

- 81,000+ planning applications
- 342,000+ associated documents
- 24 parishes covered
- Data from 1990s to present
